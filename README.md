# CAPStudy

CAP cadet Achievement Test practice quiz generator. Live at <https://capstudy.freshskyai.com>.

Pick achievement (Curry through Spaatz) + subject (Leadership Lab or Aerospace Education) → get 10 fresh practice questions with explanations. Multiple choice, instant grading, 80% pass threshold (matches actual CAP AT). Stateless — no scores stored across sessions.

Standalone Flask app, no `freshsky_common` dependency. LLM auto-fallback chain. No PII, no member data.
